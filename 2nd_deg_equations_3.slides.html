Graphing a Parabola Continued¶

The most straigt-forward way to graph a parabola is to plot a couple of points on it, and then connect them.

Let's try to graph $f(x) = x^2 - 4$

We can find that the vertex is at $(0, -4)$, and our parabola opens upwards.

def f(x):

def makeArrow(ax,pos,function,direction):
    delta = 0.0001 if direction >= 0 else -0.01
    ax.arrow(pos,function(pos),delta,function(pos+delta)-function(pos),head_width=0.2,head_length=0.5, color="black")

True or False: we always have to manually compute both the left and right side of the vertex.

False! The parabola is symmetric about $x_{vertex}$, so the value for $f(x_{vertex} - c) = f(x_{vertex} + c)$. This means we only have to compute one side of values!

Vertex Form of a Parabola¶

We have been working with parabolas with equations that look like

\begin{equation} f(x) = ax^2 + bx + c \end{equation}

But there is a way to re-arrange our terms to get our expression into vertex form.

Given a parabola, we can find

\begin{equation} (x_{vertex}, y_{vertex}) = (h, k) \end{equation}

We can then write our parabola as

\begin{equation} f(x) = a(x-h)^2 + k \end{equation}

This makes the symmetry of the parabola easy to see!

Vertex Form of a Parabola¶

True or False: The vertex form of a quadratic expression is easier to work with.

True! We don't have to calculate the vertex, only read it. Calculating points is also easier!
